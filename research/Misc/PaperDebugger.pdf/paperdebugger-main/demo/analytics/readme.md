# Google Analytics

This folder contains several Google Analytics exports used for demo paper's early usage statistics:

- [`analytics_Reports_snapshot.pdf`](./analytics_Reports_snapshot.pdf) – a
  snapshot of key GA dashboards with high-level KPIs such as active users,
  sessions, and top events for quick reference.

- [`analytics_Events_Event_name.pdf`](./analytics_Events_Event_name.pdf)
  (plus the numbered variants) – raw event reports capturing each tracked
  event and total trigger counts to study user behavior flows.

- [`analytics_Events_Event_name_since_may.pdf`](./analytics_Events_Event_name_since_may.pdf) –
  event summary since May.

- [`analytics_general_Events_Event_name_full.pdf`](./analytics_general_Events_Event_name_full.pdf) –
  full-length export of the events table, including metadata columns that
  do not appear in the lighter reports.

- [`analytics_active_users_Demographic_details_Language.pdf`](./analytics_active_users_Demographic_details_Language.pdf) – User's language distribution of active users to understand localization needs.

- [`analytics_text_patch_Events_Event_name.pdf`](./analytics_text_patch_Events_Event_name.pdf) –
  text-patch usage data.

- [`extension_active_users_Demographic_details_Country.pdf`](./extension_active_users_Demographic_details_Country.pdf) –
  country-level breakdown of active users for geographic insights.
